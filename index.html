HTML

  • Elements vs. Tags
    1. Elements includes everything written between the start and end.
    2. Tags are the literal code to at the beginning and end of an element.
  • The head element contain metadata (information needed to for the webpage).
  • The meta element contains information needed for the webpage. This content is not visible to the user (except the title which will show up on the tab).
  • The body element represents the visible content shown to the user.
  • The img element allows a developer to insert an image visable to the user.
    1. src (source) allows the developer to give the precise location of the image for insertion into the webpage.
    2. alt is important because it provides descriptions of images for screen reader apps. Accessibility is key.
